# ADR 0002: 管理平台入口統一使用 `/portal`

## 狀態

Accepted

## 背景

本專案需要為管理平台保留一條固定且清楚的入口路徑。

先前曾評估使用 `/admin`，但 `/admin` 屬於 Azure Functions 的保留路徑，無法作為可靠的應用程式入口使用。

## 決策

- 管理平台入口統一改為 `/portal`
- 程式碼、文件與後續對外設計都使用同一條 `/portal` 路徑
- 不再保留 `/admin` 作為管理平台入口命名

## 影響

- 可避免 Azure Functions runtime 保留路徑衝突
- 路徑命名更短，且本機、文件、程式碼與未來對外入口一致
