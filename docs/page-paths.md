# 頁面路徑定義

本文件目前只定義會直接由瀏覽器打開的頁面路徑。

## 目前保留頁面

| Method | Path | 說明 | 輸出格式 |
| --- | --- | --- | --- |
| `GET` | `/` | 首頁，用於供會眾生成或下載完訓證明 | `text/plain` |
| `GET` | `/verify/{certId}` | 公開驗證頁面 | `text/plain` |

## 第一版頁面內容

目前兩個頁面都只做靜態純文字顯示，不加入任何業務邏輯。

- `/`
  - 顯示頁面名稱
  - 顯示此頁用途為生成或下載完訓證明
  - 顯示目前尚未串接實際功能
- `/verify/{certId}`
  - 顯示頁面名稱
  - 顯示 `certId`
  - 顯示目前尚未串接實際驗證資料

## 暫時不保留

除上述兩個頁面外，其餘 API、管理路由與相關實作暫時移除。
